# dev-java-web-gradle-spring-thyme-couchbase-simple

## Description
A POC for springboot web app with thymeleaf support
connecting to a database.

## Tech stack
- java
- hibernate
- gradle
  - springboot
  - thymeleaf
  - couchbase connector
- bootstrap
- jquery
- dataTable

## Docker stack
- couchbase
- gradle:jdk11

## To run
`sudo ./install.sh -u`

[Available](http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
